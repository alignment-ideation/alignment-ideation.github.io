## How to Use

Each reading list is organized by topic cluster with three priority tiers:
- **Essential**: Read before judging -- core concepts all ideas build upon
- **Recommended**: Important context for specific prompts
- **Supplementary**: Deeper background, consult as needed

Resources include local file paths (for repo access) and URLs where available. Prioritize Anthropic sources and well-known AI safety labs.
